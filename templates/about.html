About AI Krishi Sahayak

Mission

AI Krishi Sahayak is an intelligent agricultural assistant designed to help farmers identify and treat plant diseases quickly and effectively using cutting-edge AI technology.

Technology

  • Gemini AI: Advanced vision analysis for disease detection
  • Open-Meteo API: Real-time weather data integration
  • Microsoft Agent Framework: Multi-agent orchestration
  • SQLite Database: Persistent data storage

Multi-Agent Architecture

Vision Agent

Analyzes plant images using Gemini AI

Research Agent

Finds treatment info & weather data

Advisory Agent

Creates farmer-friendly action plans

Memory Agent

Tracks history & schedules follow-ups

GitHub Models Agents Hackathon

This project is built for the Agents for Good track, demonstrating how multi-agent AI systems can solve real-world agricultural challenges.
